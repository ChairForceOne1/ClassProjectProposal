Application Overview

Quizmaster is a great web app to help me study for numerous subjects such as math, geography, science, and even entertainment. I can even create my own quizzes to help me study for a certain unit test. Im also able to take other peoples custom quizzes if they had made them public.

Technology Overview

To start, we used github to manage all of our code, which was programmed in html and css, and mostly in javascript. We used react library and hosted the webpage using azure. For the questions we used google firebase to store all of the questions.
